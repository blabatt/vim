\section{Visual Mode}
\textit{Entering visual mode:
\begin{itemize}
    \item v ... enter visual mode
    \item V ... enter line-visual
    \item <c-V> ... block visual
\end{itemize}
Actions d, y, c, g, J, !, \textasciitilde, all work on selections in visual mode.}
